# 球鞋商城应用

一款使用 Flutter 开发的球鞋商城应用。

## 功能特性

- 🛒 商品浏览与分类筛选
- 💰 人民币定价显示
- 🛍️ 购物袋管理
- 👤 个人中心
- ✨ 流畅动画效果

## 快速开始

```bash
# 获取依赖
flutter pub get

# 运行应用
flutter run
```

## 项目结构

```
lib/
├── animation/      # 动画组件
├── data/           # 示例数据
├── models/         # 数据模型
├── theme/          # 主题样式
├── utils/          # 工具类
├── view/           # 页面视图
├── widget/         # 通用组件
└── main.dart       # 入口文件
```
